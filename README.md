Проект автотестов для сервиса заказа самокатов

Описание
Тесты для проверки функционала сайта заказа самокатов (https://qa-scooter.praktikum-services.ru/).  
Тесты покрывают:
- Развертывание и проверку FAQ
- Навигацию по логотипам
- Полный флоу заказа через верхнюю и нижнюю кнопку заказа

- Python 3.13
- Selenium WebDriver
- Pytest
- Allure Framework
